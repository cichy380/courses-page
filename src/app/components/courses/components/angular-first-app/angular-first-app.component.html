Dla kogo

Kurs przeznaczone jest dla osób posiadających podstawową wiedzę w zakresie JavaScript, którzy chcą poszerzyć swoje kompetencje związane z tworzeniem aplikacji internetowych działających po stronie przeglądarki (SPA).

Wymagania

  • Podstawowa znajomość JavaScript, HTML oraz CSS.
    • Knowledge of TypeScript is helpful, but not required.
  • własny komputer z kartą sieciową
    Trenerzy będą mieli zainstalowane systemy Windows 10.

Oprogramowanie

Przygotowania do szkolenia przebiegnie sprawniej jeżeli uczestncy będą mieli już zainstalowane niezbędne oprogramowanie:

  • Przeglądarka Chrome
  • NodeJS w wersji co najmniej 10.13.0
    For more information on installing Node.js, see nodejs.org.
  • System kontroli wersji GIT oraz konto na GitHub
    Trener podczas szkolenia instalować będzie Git for Windows
  • Edytor kodu
    Trenerzy będą używali edytorów JetBrains.

Cel kursu

Celem kursu jest przedstawienie możliwościami frameworka Angular oraz zrozumienie jego fundamentalnych elementów. Wszystko to odbywa się podczas wspólnego budowania pierwszej aplikacji internetowej - SPA.

Plan szkolenia

  • Przygotowanie do pracy
