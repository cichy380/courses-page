Angular - pierwsza aplikacja

Poznaj framework Angular w praktyce budując z nami swoją pierwszą aplikację internetową działającą po stronie przeglądarki.

  • Start:
    24 maj 2021
  • Czas trwania:
    21h (3 dni × 7h)
  • Cena:
    1 600 PLN brutto
  • Tryb:
    Szkolenie stacjonarne
  • Lokalizacja:
    Łódź dokładny adres
  • Prowadzący:

Katering w postaci przerw kawowych (kawa, herbata, ciastka) oraz obiadu jest wliczony w cenę szkolenia.

Dla kogo

Kurs przeznaczone jest dla osób posiadających podstawową wiedzę w zakresie JavaScript, którzy chcą poszerzyć swoje kompetencje związane z tworzeniem aplikacji internetowych (SPA).

Wymagania

  • Podstawowa znajomość JavaScript, HTML oraz CSS.
    Znajomość TypeScript będzie pomocna, jednak nie jest niezbędna.
  • Własny komputer z kartą sieciową
    Trener będzie miał zainstalowane systemy Windows 10.

Oprogramowanie

Przygotowanie do szkolenia przebiegnie sprawniej jeżeli uczestncy będą mieli już zainstalowane niezbędne narzędzia:

  • Przeglądarka Chrome
  • NodeJS w wersji co najmniej 10.13.0
  • System kontroli wersji GIT oraz konto na GitHub
    Trener podczas szkolenia instalować będzie Git for Windows.
  • Edytor kodu
    Nasi trenerzy używają edytorów JetBrains.

Szkolenie rozpoczyna instalacja niezbędnych narzędzi, dlatego w przypadku ich braku uczestnicy pobiorą, zainstalują i skonfigurują je wspólnie z trenerem.

Cel kursu

Celem kursu jest przedstawienie możliwościami frameworka Angular oraz zrozumienie jego fundamentalnych elementów. Wszystko to odbywa się podczas wspólnego budowania pierwszej aplikacji internetowej - SPA.
